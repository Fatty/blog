Java基础-逻辑运算符与位运算符

2018-12-13
  • 语言基础

逻辑运算符

  • & 与
  • && 与 左侧为false时 右侧不执行（短路）
  • | 或
  • || 或 左侧为true 右侧不执行（短路）
  • ^ 异或
  • ! 非

位运算符

  • & 与
  • | 或
  • ^ 异或
    • 特点：一个数据对另一个数据位异或两次，该数本身不变。
    • 例：不定义第三方变量，实现两个整数变量交换。
    • 解法1：x = x + y; y = x - y; x = x - y;
    • 解法2：x = x ^ y; y = x ^ y; x = x ^ y;
  • ~ 按位取反
  • << 左移 左侧最高位丢弃，右边补齐0（左移几次就是乘以2的几次幂）

  • 右移 最高位是0，左边补0；最高位是1，左边补齐1（右移几次就是除2的几次幂）

  • 无符号右移 无论最高位是0还是1，左边补齐0

  • 三目运算符 （关系表达式） ？ （关系表达式1） : (关系表达式2)；

Java基础-byte类型的问题

2018-12-12
  • 语言基础

一句话解决问题

变量用于表示在程序中可能被改变的值。

a和b是“变量”意味着这两值不确定、可能随时变化，编译器无法检查，也无法确定是否在byte字节范围之内， 它检查不了就会报错（可能丢失精度），如果是“常量”编译器就能够判断了。
当两个数是常量时才进行判断常量是否在byte类型范围内：若在，就相加；若不在，就报错，因为可能会丢失精度。
int类型为java默认类型 及时损失精度 也会正常输出

Java基础-命名规则

2018-12-12
  • 语言基础

包

其实就是文件夹 用于解决相同类名问题
包名要全部小写 一般用公司域名倒着写
com.xxxx.www

类或接口

一个单词或多个单词分别举例
如果是一个单词，要求首字母大写
如果是多个单词，要求每个单词首字母大写（驼峰命名）
Adapter
ArrayAdapter

方法和变量

如果是一个单词，每个字母都小写
如果是多个单词，从第二个单词开始首字母大写
test（）
testFunction（）

常量

如果是一个单词，所有字母大写
如说是多个单词，也是所有字母大写，但是用_分开
MAX
MAX_VALUE

Java基础-基本数据类型

2018-12-12
  • 语言基础
  • 基本数据类型（4类8种）
    • 整数型
      • byte 占一个字节 -128 到 127
      • short 占两个字节 -2^15 到 2^15-1
      • int 占四个字节 -2^31 到 2^31-1
      • long 占八个字节 -2^63 到 2^63-1
    • 浮点型 默认为double （浮点型为近似表示法，无法准确等于某个小数，只能是无限逼近，尽量避免浮点型之间的比较，比较时可以引入BigDemical，但会降低效率。
      所以，要比较浮点数是否相等，需要做的事情是：1.排除NaN和无穷 2.在精度范围内进行比较）
      • float 占四个字节 1位符号位 8位指数位 23位尾数或有效数字（基于IEEE754）
      • double 占八个字节 1位符号位 11位指数位 52位尾数或有效数字（基于IEEE754）
    • 字符型
      • char 占两个字节 0~65535
    • 布尔型
      • boolean
        • boolean 理论上是占八分之一个字节，一个开关可以决定TRUE或False，但是Java中boolean 类型没有明确指定大小
