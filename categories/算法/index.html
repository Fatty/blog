剑指offer-二叉树深度

2018-12-21
  • 算法

题目描述

输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

解题思路

使用递归深度遍历二叉树，每次取深度更高的子树。

/**

public class TreeNode {
int val = 0;
TreeNode left = null;
TreeNode right = null;

public TreeNode(int val) {
    this.val = val;

}

}
*/
public class Solution {
public int TreeDepth(TreeNode root) {
if(root==null){
return 0;
}
int a = TreeDepth(root.left);
int b = TreeDepth(root.right);
return 1+((a>=b)?a:b);
}
}

剑指offer-数字在排序数组中出现的次数

2018-12-21
  • 算法

题目描述

统计一个数字在排序数组中出现的次数。

解题思路

使用二分法将数组切成多个小块，判断块中数字是否符合条件，在符合条件的区域内累加结果，不符合条件返回0。

public class Solution {
public int GetNumberOfK(int [] array , int k) {
    if(array.length==0) {
        return 0;
    }
    int num = FindByHalf(array,0,array.length-1,k);
    return num;
}
public int FindByHalf(int [] array,int l,int r,int k){
    if( l < 0 || r > array.length-1)
        return 0;
    if(array[l] > k || array[r] < k || l > r )
        return 0;
    if(array[l]==array[r] && array[l]==k){
        return r-l+1;
    }else if(array[l]==array[r] && array[l]!=k){
        return 0;
    }
    int mid = (l+r)/2; 
    return FindByHalf(array,l,mid,k)+FindByHalf(array,mid+1,r,k);

}

}

剑指offer-链表第一个公共节点

  • 算法

题目描述

输入两个链表，找出它们的第一个公共结点。

解题思路

循环遍历两个链表，设左右两个链表各有m,n个节点，总共循环遍历p次，p为m,n的最小公倍数

public ListNode FindFirstCommonNode(ListNode pHead1, ListNode pHead2) {
    ListNode p1 = pHead1;
    ListNode p2 = pHead2;
    while(p1 != p2){
        p1=(p1==null)?pHead2:p1.next;
        p2=(p2==null)?pHead1:p2.next;
    }
    return p1;
}

剑指offer-数组中的逆序对

2018-12-19
  • 算法

题目描述

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组,求出这个数组中的逆序对的总数P。并将P对1000000007取模的结果输出。 即输出P%1000000007

解题思路

利用归并排序原理，将数组用分治法排序，在排序过程中，每一次调换位置逆序数增加mid-i+1（mid为左侧归并的数组长度，i为当前数字调换后的位置，+1为数字本身占一个位置），最终加上所有的逆序数，得到最终结果

public class Solution7 {
private int base = 1000000007;
private int count = 0;
public int InversePairs(int [] array) {
    if(array.length==0) {
        return 0;
    }else {
        sort(array,0,array.length-1);
        return count;
    }
}
private void sort(int [] array,int start,int end) {
    if(start==end)
        return;
    int mid = (start+end) >> 1;
    sort(array,start,mid);
    sort(array,mid+1,end);
    merge(array,start,mid,end);
}
private void merge(int[] array,int start,int mid,int end) {
    if(start==end) {
        return;
    }
    int temp[] = new int[end-start+1];
    int i = start;
    int j = mid+1;
    int k=0;
    while(i<=mid&&j<=end) {
        if(array[i]<=array[j]) {
            temp[k++] = array[i++];
        }else {
            temp[k++] = array[j++];
            count += mid-i+1;//j在mid右侧已排好序，进入左侧后，
                 //说明这个数字之前的逆序数为左侧数组中i后面的数字个数
            count %= base;//每一步 取模防止溢出
        }
    }
    while(i<mid) {
        temp[k++]=array[i++];
    }
    while(j<end) {
        temp[k++]=array[j++];
    }
    for(k = 0; k < temp.length; k++) {
        array[start+k]=temp[k];
    }
}

}

    `import java.util.LinkedHashMap;

题目描述

把只包含质因子2、3和5的数称作丑数（Ugly Number）。例如6、8都是丑数，但14不是，因为它包含质因子7。 习惯上我们把1当做是第一个丑数。求按从小到大的顺序的第N个丑数。

解题思路

为了求取丑数数组，所有的丑数都可以由同一个公式取得：
f(x,y,z) = (2^x)(3^y)(5^z)(x,y,z为自然数)
每次结果参差不齐，所以采用三个队列存储当前最新得到的丑数。
队列t2存乘2的结果
队列t3存乘3的结果
队列t4存乘5的结果
数列中存储当前序列的丑数，一个丑数必定由另一个丑数乘2，3，5取得，因此，开始情况下t2,t3,t5三个队列的队头都为1，t2,t3,t5都为指针指向当前的位置（数组情况下是下标），当前数值被取走后，自动指向下一个数字并乘以对应的值，取得新队头，三个队头进行比较，每次取最小的队头存入丑数序列。

剑指offer-求1到N中1的个数问题

2018-12-17
  • 算法

题目描述

ACMer希望你们帮帮他,并把问题更加普遍化,可以很快的求出任意非负整数区间中1出现的次数（从1 到 n 中1出现的次数）

解题思路

个人思路：
按位计算1出现的次数，求取每个数字当前位数最大时的包含1的次数，即，0~99中共出现110+10次1，即包含十个个位数为1的数字和十个十位数为1的数字，11为重复项，但11包含两个1可以重复使用，因此0~99中数字1出现20次，依此类推，0~999包含（110+10）*10+100个1，因为百位上的1出现的次数为100次，因此可以计算0~repeat(9,n)中1个数。
因此从低位开始取，每一位中作如下几种判断

  • 当前位置为1时，看前一位是否为0，不为0的情况下，sum++(1出现的次数)，为0时不补。并且后面所有的低位数都对应一个带有1的数字，因此要全加上。
  • 当前位置大于1时，直接将Math.pow(10,j)取满，j为当前位数下所有1开头的j+1位数

  • 当前位置为0时，不作处理，留给下一位
    `import java.util.ArrayList;
    public class count_one {

网上思路：

  • 主要思路：设定整数点（如1、10、100等等）作为位置点i（对应n的各位、十位、百位等等），分别对每个数位上有多少包含1的点进行分析
  • 根据设定的整数位置，对n进行分割，分为两部分，高位n/i，低位n%i
  • 当i表示百位，且百位对应的数>=2,如n=31456,i=100，则a=314,b=56，此时百位为1的次数有a/10+1=32（最高两位0~31），每一次都包含100个连续的点，即共有(a%10+1)*100个点的百位为1
  • 当i表示百位，且百位对应的数为1，如n=31156,i=100，则a=311,b=56，此时百位对应的就是1，则共有a%10(最高两位0-30)次是包含100个连续点，当最高两位为31（即a=311），本次只对应局部点00~56，共b+1次，所有点加起来共有（a%10100）+(b+1)，这些点百位对应为1当 i表示百位，且百位对应的数为0,如n=31056,i=100，则a=310,b=56，此时百位为1的次数有a/10=31（最高两位0~30）
  • 综合以上三种情况，当百位对应0或>=2时，有(a+8)/10次包含所有100个点，还有当百位为1(a%10==1)，需要增加局部点b+1
  • 之所以补8，是因为当百位为0，则a/10==(a+8)/10，当百位>=2，补8会产生进位位，效果等同于(a/10+1)

输入n个整数，找出其中最小的K个数。例如输入4,5,1,6,2,7,3,8这8个数字，则最小的4个数字是1,2,3,4,。

解题思路

给一个数组，返回它的最大连续子序列的和，你会不会被他忽悠住？(子向量的长度至少是1)

解题思路

动态规划问题解题思路 O（n）

`public class Solution {
public int FindGreatestSumOfSubArray(int[] array) {
    int result = array[0];
    int sum = array[0];
    for(int i = 1; i < array.length; i++){
        if(array[i] + sum > result){
            result = sum + array[i];
        }
        if(sum + array[i] < 0){
            sum = 0;
            continue;
        }
        sum = sum+array[i];
    }
    return result;
}

}`
