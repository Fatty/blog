剑指offer-数组中的逆序对

2018-12-19
  • 算法

题目描述

在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组,求出这个数组中的逆序对的总数P。并将P对1000000007取模的结果输出。 即输出P%1000000007

解题思路

利用归并排序原理，将数组用分治法排序，在排序过程中，每一次调换位置逆序数增加mid-i+1（mid为左侧归并的数组长度，i为当前数字调换后的位置，+1为数字本身占一个位置），最终加上所有的逆序数，得到最终结果

public class Solution7 {
private int base = 1000000007;
private int count = 0;
public int InversePairs(int [] array) {
    if(array.length==0) {
        return 0;
    }else {
        sort(array,0,array.length-1);
        return count;
    }
}
private void sort(int [] array,int start,int end) {
    if(start==end)
        return;
    int mid = (start+end) >> 1;
    sort(array,start,mid);
    sort(array,mid+1,end);
    merge(array,start,mid,end);
}
private void merge(int[] array,int start,int mid,int end) {
    if(start==end) {
        return;
    }
    int temp[] = new int[end-start+1];
    int i = start;
    int j = mid+1;
    int k=0;
    while(i<=mid&&j<=end) {
        if(array[i]<=array[j]) {
            temp[k++] = array[i++];
        }else {
            temp[k++] = array[j++];
            count += mid-i+1;//j在mid右侧已排好序，进入左侧后，
                 //说明这个数字之前的逆序数为左侧数组中i后面的数字个数
            count %= base;//每一步 取模防止溢出
        }
    }
    while(i<mid) {
        temp[k++]=array[i++];
    }
    while(j<end) {
        temp[k++]=array[j++];
    }
    for(k = 0; k < temp.length; k++) {
        array[start+k]=temp[k];
    }
}

}
