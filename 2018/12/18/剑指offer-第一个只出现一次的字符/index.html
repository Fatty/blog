剑指offer-第一个只出现一次的字符

2018-12-18
  • 算法

题目描述

在一个字符串(0<=字符串长度<=10000，全部由字母组成)中找到第一个只出现一次的字符,并返回它的位置, 如果没有则返回 -1（需要区分大小写）

解题思路

  • 看到找只出现一次的字符，我第一时间想到用Arraylist存，遇到新字符存入，遇到重复字符去掉，但这样只有在重复字符成对出现时才成立，因此舍弃此想法。
  • 苦思无果后寻求网上的帮助，网上的方法是采用存储键值对的方式（HashMap,LinkedHashMap,HashTable,TreeMap），每当出现一次重复字符，重复字符对应的值加一。最后通过重新遍历数组，读取键对应的值的方式取得第一个出现的单独字符。
    `import java.util.LinkedHashMap;
    public class Solution6 {
    public int FirstNotRepeatingChar(String str) {
    char[] c = str.toCharArray();
LinkedHashMap<Character,Integer> map = new LinkedHashMap<Character,Integer>();
for(int i = 0; i < c.length; i++){
    if(map.containsKey(c[i])) {
        int value = map.get(c[i]);
        map.put(c[i], ++value);
    }else {
        map.put(c[i], 0);
    }
}
int loc = -1;
for(int i = 0; i < c.length ;i++) {
    if(0==map.get(c[i])) {
        loc =  i;
        break;
    }
}
return loc;
    }
    }`
