剑指offer-找数组最小k个数

2018-12-14
  • 算法

题目描述

输入n个整数，找出其中最小的K个数。例如输入4,5,1,6,2,7,3,8这8个数字，则最小的4个数字是1,2,3,4,。

解题思路

冒泡排序 太慢 肯定不行
剩下两种思路 ：

  • 分治法（快排的一部分） O（nlogn）

    • `import java.util.ArrayList;
      public class Solution {
      public ArrayList GetLeastNumbers_Solution(int [] input, int k) {
      ArrayList list = new ArrayList();
      if(input.length < k){
      return list;
      }else{
      //思路：此时采用分治法中选取中轴为k input[k-1]左侧小于右侧
exchange(input,k,0,input.length-1);
k--;
while(k >= 0){
    list.add(input[k]);
    k--;
}
return list;
      }
      }
      public void exchange(int [] input, int k,int l,int r){
      if(l>=r||l<0||r>input.length-1){
      return;
      }
      int i = l;
      int j = r;
      int temp = input[i];
      while(i<j){
      while(i<j && input[j] >= temp) j--;
input[i] = input[j];
while(i<j && input[i] <= temp) i++;
input[j] = input[i];
      }
      input[i] = temp;
      if(i == k){
      return ;
      }else if(i < k){
      exchange(input,k,i+1,r);
      }else if(i > k){
      exchange(input,k,l,i-1);
      }
      return;
      }
      }

  • 大根堆（Java中优先队列是大根堆，初始化一个容量为k的大根堆，每次读入一个数据与根对比，小数入队，大数出队） O(nlogk)

    • `import java.util.ArrayList;
      import java.util.PriorityQueue;
      import java.util.Comparator;
      public class Solution2 {
      public ArrayList GetLeastNumbers_Solution(int[] input, int k) {

      if(k > length || k == 0){
      return result;
      }
      PriorityQueue maxHeap = new PriorityQueue(k, new Comparator() {
      public int compare(Integer o1, Integer o2) {
    return o2.compareTo(o1);

      for (int i = 0; i < length; i++) {
      if (maxHeap.size() != k) {
    maxHeap.offer(input[i]);
} else if (maxHeap.peek() > input[i]) {
